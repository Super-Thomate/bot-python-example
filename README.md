Garden is a skeleton discord bot using discord.py.

API doc : <https://discordpy.readthedocs.io/en/latest/>

Install discord.py :

```
# Linux/OS X
python3 -m pip install -U discord.py

# Windows
py -3 -m pip install -U discord.py
```
